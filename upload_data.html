Upload and join your data in the underwater noise register.

To upload your underwater noise data, please, follow the next stemps

  1. 1Download the "Underwater noise register template for the Mediterranean Region" (excel format).
  2. 2Fulfil the required fields of the template following the instructions inside.
  3. 3Transform the excel file in a XML file following the instructions of the template.
  4. 4Save the XML file generated.
  5. 5Upload the generated XML file using the upload box below.
  6. 6If the upload process is successful, the application will show the message "Data uploaded successfully"*.
  7. 7Your data are printed in the map.
*If an error occurs during the upload procedure the application will inform about it. Any errors must be solved to submit the data correctly.
